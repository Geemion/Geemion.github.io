win10正式版简单体验

安装过程

去年10月份Windows10预览版刚出来就简单在虚拟机中体验过了，当时感觉就不错。即使用惯了win7的人也能很快上手，没有用win8那么bug学习代价小。正式版出来了就想马上升级试试，但是想着可能存在的兼容性问题就等了两天看了网上反馈都还不错。所以昨天就把笔记本的win7系统升级了。
这里说下笔记本配置：

  • 宏碁的性价比电脑，2010年的4741G现在用着是满目疮痍
  • I3的，6G内存（自己淘宝100块买的2手4G条子，体验还不错），GT330的显卡
    总的来说配置将就，用着win7也还好。但是本着作为搞IT的不能落后吧就升了。升级过程这里简单说下。
  • 我是下好的win10镜像升级的传送门版本是Windows 10 (Multiple Editions) (x64) - DVD (Chinese-Simplified)
  • 下好解压直接运行解压包中的setup.exe，就和安装一个普通软件一样
  • 然后就是“选择、同意”这些安装过程（注意这里有选择是否更新，注意不要选择更新不然可能等待n久）
  • 经过两个多小时等待，基本上就装完了。然后发现以前的软件都还能正常运行符合预期只是分辨率不对，要重新装哈显卡驱动
    这样就安装完成了，比较便捷。这里说哈这样安装的好处，据我所知现在win10还没有激活工具但是能免费升级（我原来win7也是D版的），但是在线升级可能这两天微软的服务器可能有点繁忙毕竟这么多人win10也4个多G，所以我是迅雷下好再升级的。但是这种方式必须满足64位win7才能升64位win10，因为那个setup.exe是64位程序。

使用感受

体验总的来说还不错，界面这些看着比较舒服、用着也没有不适感（就是跟你想的一样）。

  • 新的浏览器Edge不错，是真心不错、快、省心。但是因为我的数据配置都是chrome上应该不会当作主要浏览器。
  • 开关机比win7快很多，很好
  • 软件兼容性很好，几十年前的VC6还能很好的运行。只是对于要加载驱动的比如说像pchunter还不能运行
  • 新任务管理器很好，能很好表现电脑灯的运行状态
  • 任务切换很炫酷，用着很爽

这里推荐一些常用快捷键

  • Windows + X：快速打开快捷菜单。
  • Windows + R：快速打开运行。
  • Windows + 方向键：用于快速分屏功能。
  • Alt + Tab仍可以切换当前打开的应用程序窗口，缩略图非常大
  • Win + Tab可以打开任务视图(多桌面视图)，点击底部的“添加桌面”新建桌面，鼠标停留可以预览不同桌面，点击就可以切换

把系统中能够提高体验的配置也备注哈

  • Clover为explore提供了一个类似chrome的集成，再也不用一个个找文件夹了
  • Everything找文件必备
  • Fences桌面图表管理
  • Teamview远程管理必备
  • QQ轻聊版、QQ拼音纯净版、QQ影音、360压缩、腾讯电脑管家、Chrome、世界之窗、sublime3
  • 右键扩展复制文件文件夹路径传送们,有可能会出现乱码就自己到注册表改吧

就这么多，很好很省心。但是就是不知道为什么，升级后电脑温度有点高

更新——–使用过程中的问题

1.[20150809更新]今天突然发现启动电脑后一直出不了桌面直接黑屏，之前也看到过关于升级win10后出现黑屏的问题然后就搜了哈在这看到Windows10/8开机都黑屏，不显示桌面怎么办。Ctrl+Alt+Delete调出任务管理器进程列表中确实没有explore进程，就按照这上面的方式测试了哈。具体如下

  • 重启，万能的解决方案 重启了两次貌似不行explore进程还是没有启动
  • 手动启动explore，具体就是调出任务管理器 文件—-运行新任务—-输入explore 然后一直报错貌似是0xc0000018然后继续baidu出现一个解决方案运行explorer.exe提示0xc0000018解决方法 删除一个注册表项 嗯，还是不行。。。。。。。
  • 已几乎绝望，突然想到昨晚好像貌似点了电脑管家的优化启动项 作为一个有点强迫症的人点了，想想可能是这个问题然后手动调出[方法和上条类似，使用任务管理器运行新任务功能我的电脑管家路径是C:\Program Files (x86)\Tencent\QQPCMgr\11.0.16733.207\QQPCTray.exe] 然后找到优化项然后恢复重启。嗯，正常了。。。。。。。。

这告诉我们什么电脑管家完美兼容win10这话说得有点大了，现在win10正式版毕竟没出多久所以还是不要太强迫症了
